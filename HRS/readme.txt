The purpose of the project is to automate the cumbersome process of making a hotel reservation as well as the process of updating the hotel database with the available rooms and facilities. The project automates the following:

Operations performed by the System Administrator:
  Adding a room to the existing list of available rooms
  Deleting a room from the existing list of rooms
  Adding a customer
  Adding a food-item to the menu
Operations performed by a customer:
  Reserving a new room from the list of available rooms
  Cancelling an already booked room
  Querying information on an existing reservation
  Search Menu
  Food Ordering
